Apresentação da Atividade

Este projeto consiste em um catálogo interativo e visual de todos os componentes padrão do framework Bootstrap v5.3, desenvolvido como parte da atividade acadêmica proposta. O conteúdo foi estruturado em uma página única, onde os componentes são agrupados em seções temáticas baseadas em sua funcionalidade, como "Navegação", "Exibição de Conteúdo" e "Feedback".

A própria navegação lateral é um exemplo prático, utilizando os componentes List Group e Scrollspy para guiar o usuário pela página.

Exemplo 1: Accordion Padrão

Apenas um item pode ser aberto por vez, graças ao atributo data-bs-parent.

Um accordion é um componente de interface que permite exibir e ocultar seções de conteúdo. É extremamente útil para organizar grandes volumes de informação em uma página de forma compacta, como em seções de FAQ (Perguntas Frequentes), menus de navegação ou formulários com várias etapas.

Ele é construído com o plugin "Collapse" do JavaScript do Bootstrap. Cada item (.accordion-item) consiste em um cabeçalho (.accordion-header) e um corpo (.accordion-collapse). O botão dentro do cabeçalho controla a visibilidade do corpo correspondente através dos atributos data-bs-target (que aponta para o ID do corpo) e aria-controls.

Você pode inserir qualquer tipo de conteúdo HTML dentro do .accordion-body, como listas, imagens, parágrafos e até mesmo outros componentes do Bootstrap.

  • Tabelas
  • Imagens
