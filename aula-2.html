Apresentação da Atividade

Este projeto consiste em um catálogo interativo e visual de todos os componentes padrão do framework Bootstrap v5.3, desenvolvido como parte da atividade acadêmica proposta. O conteúdo foi estruturado em uma página única, onde os componentes são agrupados em seções temáticas baseadas em sua funcionalidade, como "Navegação", "Exibição de Conteúdo" e "Feedback".

A própria navegação desta página é um exemplo prático do uso dos componentes: o menu lateral utiliza um List Group numerado e é dinamicamente atualizado pela funcionalidade Scrollspy, guiando o usuário pelas diferentes seções. O objetivo é não apenas listar, mas também demonstrar a aplicação e a versatilidade de cada ferramenta que o Bootstrap oferece para a construção de interfaces web modernas e responsivas.
